# DevLife Simulator

🎮 Кроссплатформенная MMO игра про карьеру разработчика — от Hello World до Hello Billion

## 🚀 Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Только web клиент
npm run web

# Только сервер
npm run server
```

## 📦 Структура проекта

```
devlife-sim/
├── packages/
│   ├── web/          # Next.js веб-клиент
│   ├── server/       # Node.js API сервер
│   ├── shared/       # Общие типы и утилиты
│   └── telegram/     # Telegram Mini App (скоро)
├── docs/
│   ├── GDD.md                 # Game Design Document
│   ├── GDD-ONLINE.md          # MMO механики
│   ├── UI-UX.md               # UI/UX документация
│   ├── ART-DESIGN.md          # Арт-дизайн
│   └── TECHNICAL-ARCHITECTURE.md  # Техническая архитектура
└── README.md
```
